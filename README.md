# OCR

## Compilation

````bash
autoreconf --install
./configure
make
```

## Tâches

- Pré-traitement :
    * Niveau de gris
    * Binarisation (Otis par exemple)
    * Flou gaussien

- Découpage :
    * Découper paragraphes
    * Découper les lignes
    * Découper les caractères

- Analyse :
    * Réseau de neurones
    * Apprentissage

- Interface graphique :
    * GTK
    * Glade pour concevoir l'interface

- Site web
